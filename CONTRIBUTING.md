# Contributing

Thanks for your interest in this project!

This is a **personal project** that I maintain in my spare time. I review issues and pull requests when I have time — please be patient.

## Reporting Issues

If something doesn't work, open an issue with:
- What you expected to happen
- What actually happened
- Your OS (Windows, macOS, Linux) and how you're running Claude Code

## Pull Requests

PRs are welcome, but please:
1. **Open an issue first** to discuss the change before putting in the work
2. Keep changes focused — one feature or fix per PR
3. Test the setup script on your platform before submitting
4. Don't modify placeholder strings (`PLACEHOLDER_PATH`, `PLACEHOLDER_PLATFORM`) — those are replaced by `setup.sh` at install time

## What I'm Likely to Accept

- Bug fixes in the setup script or workflow files
- Platform compatibility improvements (especially macOS/Linux)
- Typo and clarity fixes in documentation

## What I'll Probably Decline

- Major architectural changes to the workflow steps without prior discussion
- Adding external dependencies to the setup script
- Changes that break Windows compatibility
